# Assignment: Arrays in Cairo

## Problem Statement
Write a Cairo program that creates an array of numbers, calculates the sum of its elements, and prints the result.

## Requirements
- Declare and initialize an array of at least 5 numbers.
- Calculate the sum of the array elements.
- Print the sum.

## Sample Output
```
Array: [1, 2, 3, 4, 5]
Sum = 15
```

## Instructions
- Edit `arrays.cairo` to complete the assignment.
- Follow the TODO comments.
- Run your code using Scarb.

## How to Run
```sh
scarb run
```
